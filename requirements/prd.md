# Write a Python program with the following function:
### This is a quoting calculation program that, based on starting price, target price, maximum price reduction per quote, minimum reduction amount per quote, number of decimal places to retain for price calculations, whether price calculations follow rounding, quoting result file path, etc., finds the quoting sequence that reaches the target price the fastest;
### Supports parameter input, parameters are starting price, target price, maximum price reduction per quote (percentage), minimum reduction amount per quote, number of decimal places to retain for price calculations, whether price calculations follow rounding, quoting result file path and help, all these parameters are required parameters;
### If the target price parameter is 0, then the target price needs to be calculated based on the two parameters: maximum price reduction per quote (percentage) and minimum reduction amount per quote;
### Starting from the starting price, loop to quote, reducing the quote each time until it equals the target price. The rules followed for each loop quote are: the amount of price reduction this time cannot be less than the minimum reduction amount per quote, and the amplitude (percentage) of price reduction this time cannot exceed the maximum price reduction per quote;
### The ultimate goal of the program is to find the quoting sequence that reaches the target price the fastest;
### Note that you cannot simply adopt the strategy of reducing by the maximum reduction amplitude at each step, as this would lead to failure and exit before reaching the target price due to being unable to satisfy the minimum reduction amount constraint;
### The number of decimal places retained for the above price calculation numbers shall be based on the parameter for the number of decimal places to retain for price calculations, with a default value of 2;
### For the above price calculation numbers, when the parameter whether price calculations follow rounding is true, perform rounding processing, otherwise do not perform rounding processing;
### After the quoting calculation is completed, write the data to the quoting result file, with the file format being xlsx;
### For each calculation result, create a new sheet in the quoting result file, with the sheet name being the date the calculation was completed, in the format YYYY-MM-DD, multiple calculations on the same day are all stored in the same sheet. The calculation content in each sheet includes two parts: overview and quoting sequence. The overview part content is calculation time (format is YYYY-MM-DD hh:mm:ss), starting price, target price, maximum price reduction per quote, minimum reduction amount per quote, number of decimal places to retain for price calculations, whether price calculations follow rounding, the quoting sequence part content is the information of each round of quoting, each quoting loop occupies one row of record, the record content is: quote round number (starting from 1), starting amount for this quote, amount after this quote, price reduction amount for this quote, price reduction amplitude for this quote (percentage), the content of multiple calculations on the same day is inserted at the very front in the same sheet, ensuring that the most recent calculation content is at the front, with two blank rows separating each calculation content;
### The code must include detailed comments;
### The main program must truly call the above functions to implement the requirements;
Based on the above requirements, output a complete code version.
